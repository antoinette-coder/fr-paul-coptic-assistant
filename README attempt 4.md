# Fr. Paul's Coptic Assistant

Coptic Orthodox Sermon Preparation Assistant for Fr. Paul Baky Mikhail.

## Features
- 190 Sermons (31 general + 159 Matthew series)
- 29 Books & Church Fathers writings
- Counseling resources
- Full search functionality
- Beautiful Coptic Orthodox design
